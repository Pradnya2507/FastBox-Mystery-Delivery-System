# FastBox Mystery Delivery System

FastBox Mystery Delivery System is a Python-based delivery simulation that assigns agents to packages using proximity-based logic, tracks delivery performance metrics, and generates structured reports for evaluation and analysis.

---

## Features

Nearest-agent assignment using Euclidean distance
Deterministic delivery routing (agent → warehouse → destination)
Simulated time-based delivery delays
Per-agent performance metrics
Efficiency and average delay calculation
Automatic JSON report generation
CSV export of top-performing agent
Batch execution across multiple test cases
